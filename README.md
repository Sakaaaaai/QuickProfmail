# Quick Prof Mail

大学生・大学院生向けの、教授へのメール作成を支援するWebアプリケーションです。
テンプレート管理、変数による自動挿入、そしてGoogle Gemini APIを活用したAIメール生成機能を備えています。

## 機能概要

- **テンプレート管理**:
    - よく使うメールの定型文をテンプレートとして保存・編集・削除できます。
    - 用途に合わせてカテゴリ分け（授業、研究、進路など）が可能です。
    - お気に入り機能で頻繁に使うテンプレートにすぐにアクセスできます。
- **教授リスト管理**:
    - 教授の名前とメールアドレスを登録・管理できます。
    - メール作成時に宛先として簡単に選択できます。
- **スマート変数**:
    - `{教授の名前}`、`{自分の名前}`、`{日付}` などの変数をテンプレートに埋め込むことで、送信相手や日時に合わせて自動的に内容が置換されます。
- **AIメール生成 (Gemini API)**:
    - メールの「目的」と「文体（丁寧/カジュアル）」を指定するだけで、AIが適切なメールの件名と本文を自動生成します。
- **リアルタイムプレビュー**:
    - 変数が実際の値に置き換わった状態をリアルタイムで確認できます。
- **署名管理**:
    - 自分の署名を登録し、テンプレートに自動挿入できます。
- **ローカル保存**:
    - すべてのデータはブラウザのローカルストレージに保存されるため、会員登録不要ですぐに使えます。

## 技術スタック

- **フレームワーク**: [Next.js](https://nextjs.org/) (App Router)
- **UIライブラリ**: [React](https://react.dev/), [Radix UI](https://www.radix-ui.com/), [Lucide React](https://lucide.dev/)
- **スタイリング**: [Tailwind CSS](https://tailwindcss.com/)
- **AI**: [Google Gemini API](https://ai.google.dev/)

## セットアップ手順

### 前提条件

- Node.js (v18以降推奨)
- npm または pnpm

### 1. リポジトリのクローン

```bash
git clone <repository-url>
cd professor-email-templates
```

### 2. 依存関係のインストール

```bash
npm install
# または
pnpm install
```

### 3. 環境変数の設定

プロジェクトルートに `.env.local` ファイルを作成し、Gemini APIキーを設定してください。

```bash
# .env.local
NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
```
※ APIキーは [Google AI Studio](https://aistudio.google.com/) から取得できます。

### 4. 開発サーバーの起動

```bash
npm run dev
# または
pnpm dev
```

ブラウザで `http://localhost:3000` にアクセスしてください。

## 使い方

1.  **初期設定**: 初回アクセス時に自分の名前を設定します。また、署名設定から署名を登録することをお勧めします。
2.  **教授の登録**: 「教授リスト」タブから、連絡を取りたい教授の情報を登録します。
3.  **テンプレート作成**: 
    - 「テンプレート」タブから新規作成するか、既存のものを編集します。
    - エディタ画面で「AIで生成」ボタンを押すと、AIに下書きを作成させることができます。
4.  **メール作成**:
    - テンプレートを選択し、宛先の教授を選ぶと、プレビューに完成形が表示されます。
    - 「コピー」ボタンでクリップボードにコピーし、メーラーに貼り付けて送信してください。

## 注意事項

- 本アプリケーションはブラウザのローカルストレージを使用しています。ブラウザのキャッシュをクリアするとデータが消える可能性があるため、重要なデータは別途バックアップを取ることを推奨します。
